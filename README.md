# Comparador de Preços com Selenium

Este projeto automatiza a busca de preços de um produto em diferentes lojas online e gera uma planilha Excel com a comparação dos valores.

---

## Funcionalidades

- Busca automática de preços em sites de e-commerce
- Coleta de dados utilizando Selenium
- Comparação entre diferentes lojas
- Geração de relatório em Excel (.xlsx)

---

## Tecnologias utilizadas

- Python
- Selenium
- WebDriver Manager
- Pandas
- Microsoft Edge WebDriver

---

## Como funciona o projeto

1. O usuário define o nome do produto no código
2. O Selenium abre diferentes sites automaticamente
3. O sistema captura o preço de cada loja
4. Os dados são organizados em uma tabela
5. Um arquivo Excel é gerado com a comparação
